A CMS Project by Malachi Jones

This project is an example of a simple CMS, using the following technologies:

  • Angular 6
  • Firebase
  • Google OAuth 2.0
  • Bootstrap

Navigation and Web App Functionality

In regard to the 4 links on the navigation bar, I've developed this CMS so that all pages are accessible as a visitor, apart from the edit-posts page.

Logging In

Attempting to access the edit-posts page will prompt you to log in, this is achieved via the use of an AuthGuard class which checks if a user is signed in or not.

Google's OAuth 2.0 (importing the AngularFireAuth) is used for logging in users; once logged in, you will automatically be given access and be able to edit and add posts to the Firebase database.

The Firebase 'Firestore' Database

Using JavaScript RXJS, a connection to Google's Firebase Firestore database is made, providing realtime asynchronous feedback on all CRUD operations.
